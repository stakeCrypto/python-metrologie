## Installation

```bash
# Cloner le repo
git clone https://github.com/ton-username/python-metrologie.git

# Creer un environnement virtuel
python -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate   # Windows

# Installer les dependances (quand disponible)
pip install -r requirements.txt
```
